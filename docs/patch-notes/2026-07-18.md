# 패치노트 (2026-07-18)

시청자 피드백(GitHub 이슈 #13)을 처리한 내용과, 그 과정에서 발견해 함께 고친 온라인 대전 버그를 정리했습니다.

## 🐛 피드백 반영 - 버그 수정 / UI 개선

- **피드백 버튼과 채팅 버튼이 겹치던 문제**를 고쳤습니다. 피드백 버튼을 화면 왼쪽 아래로 옮겼습니다.
- **체크무늬 증강의 착수 가능 칸 표시**가 너무 옅어서 잘 안 보이던 문제를 고쳐, 눈에 잘 띄도록 진하게 만들었습니다.
- **게임이 끝난 뒤에도 최종 보드를 다시 볼 수 있게** 했습니다. 증강 선택 화면에 있던 "누르고 있으면 판 보기" 기능을 승리 화면에도 추가했습니다.
- **금지 구역 / 결계처럼 칸을 여러 개 선택하는 증강**에서, 잘못 고른 칸을 다시 클릭하면 선택이 취소되도록 고쳤습니다. (이전에는 한 번 고른 칸은 무를 방법이 없었습니다.)
- **"판 보기" 중에 상대 증강 패널을 클릭할 수 없던 문제**를 고쳤습니다. 이제 판을 보는 동안에도 상대가 가진 증강의 설명을 확인할 수 있습니다.
- **액티브 능력을 사용할 때 효과음**이 나도록 추가했습니다. 착수음/증강 등장음/내 턴 알림과는 다른 소리로, 능력이 실제로 발동했을 때만 재생됩니다.

## ⚖️ 밸런스 조정

- **"돌을 한 번에 2개 놓는" 계열 증강 3종(도미노, 양수겹침, 양보 없음)을 삭제**했습니다. 상대 입장에서 카운터플레이가 불가능한, 알면서도 당해야 하는 카드라는 피드백을 반영했습니다. (질풍노도는 이 원칙의 의도된 예외로 그대로 남아있습니다.)
- **재도전 시 색 선택 방식을 변경**했습니다. 이제 진 쪽이 다음 판에 흑돌/백돌 중 원하는 색을 직접 고를 수 있고, 이긴 쪽은 자동으로 반대 색을 받습니다. (기존에는 승패와 무관하게 무조건 색이 뒤바뀌었습니다.)
- **시작 증강(첫 착수 전) 선택 풀에서 제외되는 카드 범위를 넓혔습니다.** 기존에는 칸 지정형 카드 몇 개만 빠졌는데, 이제 즉시발동형 전역 효과(질풍노도, 교도소, 폭주 등)와 상대 돌 제거 계열(포위 제거, 오델로, 습격 등)까지 제외해서, 첫 수를 두기도 전에 판이 사실상 결정나 버리는 상황을 줄였습니다.
- **둔갑술 증강을 다시 설계**했습니다. 기존에는 버튼을 눌러 보유 카드 하나를 1회 위장시키는 액티브 능력이었는데, 이제 뽑는 즉시 발동해서 그 이후로 얻는 카드마다 자동으로(반복해서) 상대 화면에서만 위장됩니다.

## 🌐 온라인 대전 버그 수정

- **초대 링크로 대전 시작 시, 흑돌이 시작 증강을 먼저 고르고 백돌이 뒤이어 들어오면 백돌의 시작 증강 선택 화면이 멈추는 버그**를 제보받아 조사했습니다. 코드 검토 결과 페이지 접속 직후 "최초 상태 불러오기"와 "실시간 갱신 구독"이 동시에 시작되는데, 그 사이의 아주 짧은 시간차 안에 상대가 만든 갱신이 오면 놓칠 수 있는 레이스 컨디션을 발견해 방어적으로 수정했습니다. 실시간 구독이 확정되는 순간 최신 상태를 한 번 더 안전하게 다시 가져오도록 했습니다.
  - ⚠️ 이 수정은 실제 온라인 대전(두 명이 동시 접속)으로 아직 확인되지 않았습니다. 같은 증상이 다시 나타나면 알려주세요 - 다른 원인을 다시 조사하겠습니다.
